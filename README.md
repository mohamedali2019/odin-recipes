# Odin Recipes

This project is part of the [The odin Project] (https://www.theodinproject.com/). it's a simple HTML website that contains link to different recipes. 

## Features 
- Homepage with recipe links
- Individual recipe pages with ingredients and steps
- Pratcie with basic HTML structure

## How to Run

Clone the repo and open 'index.html' in your browser: 
```bash
git clone https://github.com/<your-username>/odin-recipes.git
cd odin-recipes
xdg-open index.html # Linux xommand to open in default browser
